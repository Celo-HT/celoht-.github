# CeloHT Infrastructure

_Last updated: July 20, 2026_

## Overview

This document describes the infrastructure principles and systems used to support CeloHT projects.

The goal is to maintain secure, reliable, scalable, and accessible technical services.

---

# Infrastructure Principles

CeloHT infrastructure should prioritize:

- Security
- Reliability
- Scalability
- Transparency
- Cost efficiency
- Maintainability

- Development tools
