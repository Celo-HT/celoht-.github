# Why CeloHT Does Not Have a Native Token

CeloHT intentionally does not create its own cryptocurrency.

Instead, the ecosystem builds upon the trusted financial infrastructure already provided by the Celo blockchain.

This decision allows the project to:

- Focus on real-world utility instead of token speculation.
- Reduce technical complexity.
- Improve long-term sustainability.
- Encourage adoption of CELO and cUSD.
- Remain fully aligned with the mission of the Celo ecosystem.

The success of CeloHT will be measured by community impact, education, innovation, and ecosystem growth—not by the market value of a proprietary token.
