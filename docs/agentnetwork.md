# CeloHT Agent Network

## Overview

The CeloHT Agent Network is a community-based initiative designed to connect people with accessible digital financial services and Web3 education.

Agents serve as local support points that help individuals and communities understand and use blockchain technology in practical ways.

---

# Mission

The mission of the CeloHT Agent Network is to reduce barriers to digital financial access by providing trusted local assistance, education, and support.

Agents help bridge the gap between blockchain technology and everyday users.

---

# Role of CeloHT Agents

CeloHT Agents support communities through:

- Digital Wallet Assistance
- User Education
- cUSD Support
- CELO Support
- Transaction Guidance
- Community Outreach
- Merchant Support
- Web3 Awareness

Agents help users gain confidence when interacting with digital financial tools.
