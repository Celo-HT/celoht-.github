# Contributing to CeloHT

Thank you for contributing to CeloHT an open-source ecosystem built for financial inclusion, education, and climate impact.

## Principles
- Impact-first  
- Security-by-design  
- Community-centric  
- Transparency & documentation  

## Contribution Types
- Code  
- Documentation  
- Research  
- Community  

## Workflow
1. Check existing issues  
2. Open a new issue using the correct template  
3. Fork → Create branch  
4. Write tests  
5. Update documentation  
6. Open Pull Request  

## Coding Standards
- Clean, maintainable code  
- Tests required  
- No breaking changes without discussion  
- Clear commit messages  

## Governance
Major changes require a proposal and review by maintainers & stewards.

## Security
Do **not** open public issues for vulnerabilities.  
Follow the Security Policy.

Thank you for helping build a world-class open-source ecosystem.
